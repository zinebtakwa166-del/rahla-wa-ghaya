
# رحلة وغاية — نسخة متعددة الصفحات (Node12 + Tailwind)

التشغيل:
1) داخل المجلد:
   npm install
2) التشغيل:
   npm run dev
3) افتح http://localhost:3000

الصفحات:
- /            صفحة البداية
- /dashboard   لوحة القيادة + بطاقات رحلات
- /trips/TRP-001 تفاصيل رحلة ديناميكية
- /contact     اتصل بنا (هاتف/بريد/واتساب)
- /payment     التسديد (بطاقة/نقدًا/تحويل)
